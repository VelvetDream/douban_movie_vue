## 开发流程

### 后端开发 (Spring)

[后端仓库](https://github.com/humingk/douban_movie_plus)

### 前端开发 (Vue)

#### 0. fork

#### 1. clone (替换用户名)

`git clone -b dev --depth 1 https://github.com/humingk/douban_movie_vue`

or

`git clone -b dev --depth 1 git@github.com:humingk/douban_movie_vue.git`

#### 2. npm安装

- npm 6.13.4

`npm install`

#### 3. 启动

`npm run serve`

#### 4. 请提交到**dev**分支
